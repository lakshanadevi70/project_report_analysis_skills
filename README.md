# 📊 Project Analysis Report Skill

An AI-powered project analysis skill that reviews software projects and automatically generates a comprehensive Markdown report with architecture insights, API analysis, performance evaluation, scalability review, code quality assessment, and improvement recommendations.

---

## 🚀 Overview

Project Analysis Report Skill is designed to help developers, testers, architects, and reviewers understand a software project quickly.

Instead of manually reviewing hundreds of files, this skill analyzes the project structure and generates a detailed report highlighting:

- Project architecture
- API analysis
- Performance review
- Scalability
- Validation checks
- Error handling
- Repository flow
- Complexity analysis
- Improvement suggestions

The final output is generated as a well-structured **Markdown report**.

---

# ✨ Features

## 📁 Project Structure Analysis

- Detects project folders and files
- Identifies application layers
- Explains project architecture
- Finds important modules

---

## 🔍 API Analysis

Automatically analyzes APIs and identifies:

- Internal APIs
- External APIs
- API ownership
- HTTP methods
- Endpoints
- Request flow
- Response handling

---

## ⚡ Performance Analysis

Evaluates:

- API response efficiency
- Database access patterns
- Code optimization
- Expensive operations
- Performance bottlenecks

---

## 📈 Scalability Review

Checks whether the application is scalable by analyzing:

- Modular architecture
- Service separation
- Layered design
- Dependency management
- Code organization

---

## ✅ Validation Analysis

Reviews validation practices including:

- Request validation
- Input validation
- Required field validation
- Business rule validation

---

## 🚨 Error Handling Analysis

Detects:

- Exception handling
- Try-Catch blocks
- API error responses
- Logging strategy
- Error consistency

---

## 🏗 Repository & Service Flow

Analyzes application flow such as:

```
Client
   ↓
Controller
   ↓
Service
   ↓
Repository
   ↓
Database
```

Provides a simple flow explanation for easier understanding.

---

## 🧩 Complexity Analysis

Identifies:

- Complex methods
- Deep nesting
- Large classes
- Duplicate logic
- Maintainability concerns

---

## 💡 Improvement Suggestions

Generates actionable recommendations for:

- Performance optimization
- API improvements
- Better architecture
- Security enhancements
- Code refactoring
- Best practices
